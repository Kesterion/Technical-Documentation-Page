What is HTML?

HTML stands for Hyper Text Markup Language and is responsible for describing the structure of a web page. HTML is made up of various elements which tell the browser how to display the content. These elements are represented by tags.

Here are some examples of html tags:

  • The !DOCTYPE declaration defines this document to be HTML5.
  • The html element is the root element of an HTML page.
  • The head element contains meta information about the document.
  • The title element specifies a title for the document.
  • The body element contains the visible page content.
  • The h1 element defines a large heading.
  • The p element defines a paragraph.

Here is an example of how html is structured:

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>

<p>My first paragraph.</p>

</body>
</html>

HTML Documents

The DOCTYPE declaration represents the document type and helps browsers display web pages correctly. This decleration should only appear once on every page before any other code.

This is the DOCTYPE declaration for HTML5:

<!DOCTYPE html>

All HTML documents must begin and end with html tags. However, bear in mind that only the content within the body tags will be visible on the webpage.

These are the html and body tags for HTML5:

<html></html>

<body></body>

HTML Headings

HTML headings are used as headings for content within a web page, their importance is indicated by the number within the tag, see below for the first three headings.

<h1>This is Heading 1</h1>
<h2>This is Heading 2</h2>
<h3>This is Heading 3</h3>
HTML Paragraphs

The main text/content within a web page is defined in a paragraph.

<p>Lorem, ipsum dolor sit amet consectetur adipisicing elit. Dolorem, sit.</p>
<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Maiores debitis fugit nobis sint sed dolores in nihil laborum alias unde!</p>
HTML Links

HTML links are defined with the a tag. The link's destination in specified within the href attribute. Attributes are used to provide additional information about HTML elements.

<a href="https://www.freecodecamp.org">This is a link to freeCodeCamp</a>
